## OP.GG for Custom Games

Blueward is a stat-tracking website built to boost engagement in and competition between university League of Legends clubs through custom games (“inhouses”).

The idea behind Blueward began as a Google Doc, where club officers in Longhorn LoL (UT Austin) would manually update club records (Kills, Damage, CS, etc.) after inhouse games. It has since evolved into a detailed tracker for all League of Legends related stats from the custom games that don't show up on any other public tracking website.

Want to learn more about Blueward? Join the [Longhorn LoL discord](https://www.longhornlol.com/).
